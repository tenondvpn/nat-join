### 1. Download the code:

      git clone https://github.com/tenondvpn/tenonvpn-join.git

      wget https://github.com/tenondvpn/tenonvpn-join/archive/1.2.0.zip

### 2. init system, system will restart after execution， node will start automatically

      bash init_env.sh
